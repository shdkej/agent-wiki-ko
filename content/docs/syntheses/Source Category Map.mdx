# 소스 카테고리 지도

## Fundamental
기술적 기초 모델. 오래 가는 엔지니어링 원칙이 많이 숨어 있을 가능성이 높다.

## Integration
기술 역량이 제품, 비즈니스, 디자인, 실무로 이어지는 연결 층.

## Communication
글쓰기, 설명, 협업, 명료성 기준을 담는 층.

## Human
가치관, 세계관, 미래 지향성, 정체성 수준의 기준을 담는 층.

## Idea
호기심, 취향, 읽기, 미디어, 탐색 신호를 담는 층.

## Meta
학습, 수집, 트러블슈팅, 개발자 규율 같은 메타 사고 층.

## blog
상대적으로 다듬어진 바깥 표현물. 내부 노트와 외부 표현의 차이를 보는 기준점이 될 수 있다.
