# 지식 랩 운영 가설

이 랩의 목적은 사용자의 장문 노트를 원본 경계를 유지한 채 더 관리하기 쉬운 고품질 위키로 계속 재구성하는 것이다.

## 핵심 가설
- 원본 노트는 사용자의 원래 사고 방식을 보존한다.
- 에이전트 위키는 더 정돈되고 연결된 지식 레이어를 보존한다.
- 나중에 들어오는 인간 수정본은 보정 신호로 작동한다.
- 드리프트는 관찰 가능하고 되돌릴 수 있고 비교 가능하다면 허용할 수 있다.

## 성공 조건
에이전트는 범용적으로 더 똑똑해지는 것이 아니라, 사용자의 반복되는 니즈, 기준, 선호 작업 방식에 더 잘 맞아가야 한다.

## 위험
- 문체 드리프트
- 과도한 추상화
- 원본 근거를 벗어난 구조 발명
- 페이지 비대화

## 제어 장치
- 500줄 제한
- 카테고리 단위 작업
- 일일 요약
- 원본 기반 재생성 가능성
